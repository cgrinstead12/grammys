Spotify Api

Utilizing the Spotify API and Genius API, our pull was for the various features from a list of songs from potential nominees.




Machine Learning Model

Using the scikit learn library’s Random Forest Classifier ensemble method, we trained the data and used it to determine the relative importance’s of each feature in our model.

The random forest model consists of several simple classification “trees” that are built from subsets of the data and these independent trees make their own classification predictions.

An independent decision tree is simply a series of decisions that are used to classify any individual record in a data set. Each of these independent prediction trees are then combined into one (an ensemble of trees), which is as good or better than any one individual classifier.

We chose the random forest model because it is capable of handling a large number of features and it is helpful in estimating which of our features are most important. In this model, “feature.speechiness” was determined our most important feature in predicting a Grammy winning song.

Results

Once we fed in our historical data to train the model and get our feature relative importance’s, the next step was to feed in our new data from the Billboard top 100 songs of 2018 into the model to see which artists had the highest probability of winning a Grammy award.

We used the random forest model’s predict method and created our new data set which lists each artist’s tracks along with their probabilities of winning. From our model we found that Maroon 5’s “Girls Like You” has the highest chance of winning the Grammy for 2018 based on our models predictions using the song features of Grammy award winners, nominees and Billboard top 100 songs from the last seven years.
